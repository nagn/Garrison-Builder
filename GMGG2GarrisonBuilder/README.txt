GarrisonBuilderIcon.pdn is the Paint.Net file used to create the GarrisonBuilder's icon.  Paint.Net is a free image editor for Windows.---gitToGmk.bat will create garrisonBuilder.gmk from the Source directory.  Thisshould be run after merging in new revisions from version control.gmkToGit.bat will split garrisonBuilder.gmk into files in the Source directory.This should be run right before committing changes to version control.These batch scripts require Medo42's Gmk Splitter.  The easy solution is tocopy gmksplit.exe into this directory.---GG2PNGSave.gex is a Game Maker extension required by the Garrison Builder.
